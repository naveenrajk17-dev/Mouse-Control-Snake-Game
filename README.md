# 🐍 Mouse Control Snake Game

An interactive animated snake game built using **HTML, CSS, and JavaScript**.

The snake follows the mouse cursor smoothly.  
Eat 30 mice to win the game!

---

## 🎮 Game Features

- 🖱 Mouse-controlled snake movement
- 🌳 Multiple forest trees as obstacles
- ⭐ Stun effect with rotating stars animation
- 🎯 Win goal: 30 mice
- 🐍 Smooth body follow mechanics
- 🖥 Large responsive game screen
- ⚡ Game Over & Win conditions

---

## 🛠 Technologies Used

- HTML5 Canvas
- CSS3
- JavaScript (Vanilla JS)

---

## 🚀 How to Play

1. Move your mouse to control the snake
2. Eat the pink mice
3. Avoid hitting walls
4. If snake hits tree:
   - Snake gets stunned
   - Stars appear above head
   - After few seconds snake can pass the tree
5. Eat 30 mice to win
